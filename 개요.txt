
네이버 재입고 알림 서비스

기본 동작
1. 재입고 체크하고 싶은 상품을 찜목록에 넣기
2. 찜 목록 항목들 전부 크롤링 후 저장
3. 이후 정해진 시간에 찜목록과 저장한 데이터 비교
    1. 품절 -> 입고 된 상품이 있다면 검사
    2. 내가 원하는 옵션이 재입고 된건지 판단
4. 원하는 옵션이 입고가 됐으면 슬랙으로 알려줌



체크할 거
품절 상품들 양식을 좀 봐보자
1. 완전 품절된경우 사이트에 표시됨
2. 일부만 품절인 경우 사이트에는 표시 안되고 목록에 품절이라고 표시됨
3. 일부만 품절인 경우 어떤 항목의 품절 여부를 확인하고 싶은지 체크해야 함
4. 데이터 저장은 엑셀로 해두면 될거 같음


일부만 품절인 경우 어떤 항목의 품절 여부를 확인하고 싶은지 체크해야 함

2. 콤보박스에서 옵션들중 뭔가 고르면 옵션 2버튼 클릭 후 클로링 후 대기
3. 콤보박스에서 옵션들중 뭔가 고르면 옵션 2버튼 클릭 후 클로링 후 대기

4. 콤보박스에서 마지막 옵션까지 고르면 고른 값들 저장
